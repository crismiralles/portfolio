Conocimientos Técnicos

1. Maquetación web con HTML y CSS

Desarrollo de estructuras semánticas con HTML5 y diseño adaptable con CSS3. Aplicación de principios responsive y buenas prácticas de accesibilidad.

  • Creación de layouts con Flexbox y Grid.
  • Uso de clases reutilizables, variables y pseudoelementos.
  • Optimización del diseño para distintos dispositivos y navegadores.
  • Integración con Bootstrap para una maquetación rápida y responsive.
2. Tipos de datos, estructuras y funciones en JavaScript
  • Tipos de datos: numéricos, cadenas, booleanos
  • Variables y operadores (aritméticos, lógicos y relacionales)
  • Estructuras de control:
    • Secuenciales: ejecución línea a línea
    • Selectivas: if, else
    • Repetitivas: for, while
  • Funciones: modularización del código en JavaScript
3. Ficheros

Conozco cómo trabajar con ficheros (lectura/escritura) usando JavaScript del lado del cliente.

4. Bases de datos

He estudiado distintos tipos de bases de datos:

  • Jerárquicas
  • Relacionales: tablas, claves primarias/foráneas, SQL
  • Documentales

También conozco el uso de SGBD (como SQLite) y el lenguaje SQL para consultas, creación de tablas, inserción de datos, etc.
